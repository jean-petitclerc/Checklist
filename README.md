# Checklist

## Intentions:

Gèrer via un interface Web codé avec Flask des checklists. Les checklists doivent permettrent des subtitutions de variables. Pour le moment, les checklists auront des sections et des étapes uniquement.

- Checklist
  - Description de la checklist
  - Description des variables
  - Section 1
    - Etape 1
    - Etape 2
    - ...
  - Section 2
  - ...
